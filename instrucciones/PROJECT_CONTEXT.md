# Project Context

## Objetivo
Sistema para automatizar la gestión de WhatsApp Business con IA.

## Stack
- Frontend: Next.js, React, TypeScript, Tailwind
- Backend: Node.js, Express, TypeScript
- Base de datos: Supabase (PostgreSQL)
- Auth: Supabase Auth
- Storage: Supabase Storage

## Principios
- Arquitectura modular.
- Escalabilidad.
- Código mantenible.
- IA como apoyo, no sustituto de la lógica de negocio.
